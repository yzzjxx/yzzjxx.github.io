JavaScript笔记

2018-08-23

Welcome to myBlod! This is my very first test post. Please give me some marks! Thanks for you reading! Have a good time!

.JavaScript介绍（原名LiveScript，作者：布兰登·艾奇）

.JavaScript是什么

  1. 是一门脚本语言
  2. 是一门解释性语言
  3. 是一门动态类型的语言
  4. 是一门基于对象的语言

编译语言：需要要代码计算机所认知的二进制语言，才能够执行

脚本语言：不需要编译，直接执行（t-sql、cmd）

.JavaScript的作用

最初：解决用户与浏览器之间的交互问题

现在：

  1. 网页特效
  2. 服务端开发（nodejs)
  3. 游戏开发
  4. APP

.JavaScript代码注意问题

  1. 在一对script标签中，有错误的JS代码会导致该错误代码后面的JS代码不执行
  2. 如果第一对的script标签中的JS代码有错误，不会影响后面的script标签中的JS代码执行
  3. script标签包含 type=“text/javascript” 是标准写法，script标签中也可以包含 language=“JavaScript”
  4. 目前的HTML页面，script中的type和language都可以省略，因为HTML遵循h5的标准
  5. script 标签一般放在 body 最后，可以提升浏览器的执行效率
  6. js 中大小写是区分的
  7. 变量命名要遵循驼峰命名法
  8. 变量是用来操作数据的（存储和读取）
  9. 单行注释 //
  10. 多行注释 /**/

. JS 数据类型

  1. number：数字类型（整数和小数）

  2. string：字符串类型（值一般用引号包含起来）

  3. boolean：布尔类型（值只有两个，true（真1），false（假0））

  4. null：空类型，值只有一个：null，一个对象指向为空，此时可以赋值为null

    *null是对象类型，表示空对象，alert(typeof(null)); 会输出object；

    *alert(null == document.getElementById(‘notExistElement’));

    ​ 当页面上不存在id为”notExistElement”的DOM节点时，这段代码显示为”true”，因为我们尝试获取 一个不存在的对象。

    *alert(null == undefined); //output “true”

    *alert(null === undefined); //output “false”

    *alert(typeof null == typeof undefined); //output “false”

  5. undefined：未定义，值只有一个：undefined

    *变量声明了，没有赋值，结果是undefined；

    *函数没有明确返回值，如果接收了，结果是undefined；

    *一个结果是undefined的变量与一个数字进行计算，结果为NaN（not a number）

  6. object：对象

. JS 数字类型

  1. 二进制 （00000001）
  2. 八进制 （00000012）
  3. 十进制 （10）
  4. 十六进制 （0x1f）
  5. 数字类型的范围：
    • console.log(Number.MAX_VALUE); //数字的最大值
    • console.log(Number.MIN_VALUE); //数字的最小值
  6. 注意：
    • 不要用小数去验证小数
    • 不要用NaN验证是不是NaN，应该使用 isNaN()
      • isNaN 中变量不是一个数字，返回true
      • isNaN 中变量是一个数字，返回false

. JS 字符串类型

  1. 字符串长度：str.length;
  2. 转义符：\
  3. 字符串拼接
    • ＋号两边有一个字符串，那么为字符串拼接
    • ＋号两边都是数字，那么为计算功能

. JS 类型转换

  1. 其他类型转为数据类型：
    • 想要转整数用 parseInt() ,想要转小数用 parseFloat()
    • 想要转数字 Number()，这个方法比上面两种方式严格
  2. 其他类型转为字符串类型：
    • str.toString() //无意义的字符串使用这个方法会报错
    • String(str) //无意义的字符串使用这个方法不会报错
  3. 其他类型转为布尔类型：
    • Boolean(value)

.操作符

  1. 算术运算符：+ - * /
    • 算术运算表达式：由算术运算符连接起来的表达式
  2. 一元运算符：只需要一个操作数就可以运算的符号（++ –）
  3. 二元运算符：需要两个操作数才能运算的符号（）
  4. 三元运算符：需要三个操作数才能运算的符号（）
  5. 复合运算符：+= -= *= /= %=
  6. 关系运算符：> < >= <= == === != !==
    • 关系运算表达式：由关系运算符连接的表达式
    • 关系运算表达式的结果为布尔类型
  7. 逻辑运算符：&& || !
    • 逻辑运算表达式：由逻辑运算符连接起来的表达式

.计算机

  1. 计算机由硬件组成，需要操作系统才能运行
  2. 应用程序（QQ、英雄联盟）在计算机的硬件中
  3. 打开应用程序时，CPU指派硬件中的应用程序到内存中执行
